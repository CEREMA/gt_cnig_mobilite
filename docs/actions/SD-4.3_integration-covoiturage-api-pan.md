## SD-4.3 : Améliorer l'intégration des offres de covoiturage au sein des applications de mobilité en réglementant l'ouverture des API et leur référencement sur le PAN

Dans le cadre de l'Action 4 "Développer les outils et pratiques numériques favorisant le report modal et le covoiturage", cette sous-action vise à développer des API ouvertes et sécurisées facilitant le partage d'informations entre acteurs publics et privés pour améliorer la visibilité, la régulation et la confiance autour du covoiturage.
