## SD-4.5 : Créer une API ouverte dédiée au contrôle de validité des permis de conduire pour sécuriser les trajets de covoiturage et le recours à l'autopartage, prévenir les fraudes aux incitations financières

Dans le cadre de l'Action 4 "Développer les outils et pratiques numériques favorisant le report modal et le covoiturage", cette sous-action vise à sécuriser les pratiques de covoiturage et d'autopartage en permettant la vérification en temps réel de la validité des permis de conduire via une API ouverte.
