## SD-5.1 : Fournir des API de vente de service de mobilité : transport public, stationnement, mobilité partagée (autopartage, vélo, trottinette en libre-service)

Dans le cadre de l'Action 5 "Fournir des infrastructures socles pour simplifier la construction des services numériques multimodaux", cette sous-action vise à mettre en place des API de vente de titres et services de mobilité pour fluidifier l'achat dématérialisé dans le cadre du projet Titre Unique.

### Porteurs
- **DGITM**
- AFNOR
- FNMS ITxPT
- Fabmob

### Calendrier
T3 2024 - 2026
